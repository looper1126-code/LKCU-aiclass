# 第11週：AI 翻譯：出國應用
日期：11/13（週五）｜授課時數：3小時

## 📖 課程摘要

### AI 讓語言不再是出國障礙

很多長輩想要出國旅遊，但最大的擔憂就是語言不通。「看不懂英文菜單怎麼辦？」「機場廣播說什麼聽不懂？」「在國外生病了怎麼跟醫生描述症狀？」這些問題，現在都可以靠 AI 翻譯工具輕鬆解決。

本週課程會帶給大家一套完整的出國翻譯攻略，從出發前的準備、機場通關、當地交通、餐廳用餐，到緊急情況處理，全部都有 AI 幫忙的方案。

### AI 翻譯的三大工具

本週我們會用到三個工具：
1. **Google 翻譯**：離線也能用，支援拍照翻譯
2. **ChatGPT**：翻譯更自然，可解釋文化差異
3. **Gemini**：結合 Google 搜尋，翻譯即時資訊

### 本週學習目標

1. 安裝並熟悉 Google 翻譯 App
2. 學會用「拍照翻譯」功能看懂外語菜單和路標
3. 演練機場、飯店、餐廳、問路的實用情境對話

## 🛠️ 實作步驟

### 步驟1：安裝 Google 翻譯

**下載 Google 翻譯：**
- Android：到 Play 商店搜尋「Google 翻譯」，下載
- iPhone：到 App Store 搜尋「Google 翻譯」，下載

**設定偏好語言：**
1. 開啟 Google 翻譯
2. 點左上角語言切換，選擇「中文（繁體）」和要去旅遊的語言
3. 例如要去日本，選「中文（繁體）」和「日語」
4. 可以同時設定多組常用語言

**開啟離線翻譯（重要！出國必備）：**
1. 設定好語言對後，點「下載」
2. 下載離線翻譯包（約 50-100MB，飯店 WiFi 先下載好）
3. 這樣在國外沒有網路時也能用翻譯功能

（請見課堂截圖：Google 翻譯設定說明）

### 步驟2：拍照翻譯——秒懂外語菜單

在國外餐廳，最怕的就是看不懂菜單。用 Google 翻譯的「相機翻譯」功能，對著菜單一拍，立刻看到中文字！

**操作步驟：**
1. 開啟 Google 翻譯
2. 確認上下語言已設定正確（例：繁體中文 ↔ 日語）
3. 點選中間的「相機」圖示
4. 對著外語文字拍照，畫面會即時顯示翻譯後的文字
5. 可以用手指滑動，選擇只翻譯特定單字或句子

**練習：**老師會準備一張日文或英文的範例菜單，請大家拿出手機對著拍攝，翻譯出來的內容應該會很有趣。

### 步驟3：機場情境翻譯演練

讓我們演練在機場可能遇到的情況：

**情境一：辦理登機手續**
```
我需要以下這句話的英文，請幫我翻譯：
「我想辦理這班飛機的登機手續，請問在這裡辦理嗎？」
```
AI 翻譯：「I would like to check in for this flight. Is this the right counter?」

**情境二：托運行李有問題**
```
請幫我把這句話翻成英文：
「這是我的行李，超重了，請問需要補多少費用？」
```

**情境三：機場廣播聽不懂**
有時廣播會呼叫你的名字或通知重要事情。遇到聽不懂的廣播，立刻打開手機，對著 AI 說：「請幫我翻譯成中文」。

### 步驟4：飯店情境翻譯

**入住登記：**
```
請幫我把這句話翻成日文/英文：
「我有預訂今天晚上的房間，名字是○○，請幫我辦理入住」
```

**遇到問題時：**
```
請幫我把這句話翻成日文/英文：
「房間的空調不冷，熱水也不夠熱，請派人來查看」
```

**延長住宿：**
```
請幫我把這句話翻成日文/英文：
「我想多住一天，明天下午再退房，請問可以嗎？」
```

### 步驟5：餐廳用餐翻譯

**看圖片點菜（最簡單的方式）：**
用手機拍下鄰桌客人點的菜，直接問 AI：
```
請幫我看看這張照片裡的菜是什麼，用中文描述，並告訴我好不好吃
```

**自己點餐：**
```
請幫我把這句話翻成日文/英文：
「我想點一份這個（手指菜單），但是我對海鮮過敏，請確認這道菜沒有海鮮」
```

### 步驟6：問路和交通翻譯

**問路時：**
```
請幫我把這句話翻成日文/英文：
「請問這個地址怎麼走？可以幫我畫個地圖嗎？或者幫我叫計程車？」
```

**搭公車或地鐵：**
```
請幫我把這句話翻成日文/英文：
「請問這班公車會到○○站嗎？我要在第幾站下车？」
```

### 步驟7：緊急就醫翻譯

在國外就醫是最令人擔心的情境之一。讓我們事先準備好萬用句子。

**說明症狀：**
```
請幫我把這句話翻成日文/英文：
「我身體不舒服，症狀是：頭痛、發燒、想吐、沒有力氣。請問最近的醫院在哪裡？」
```

**描述病史和用藥：**
```
請幫我把這句話翻成日文/英文：
「我有糖尿病和高血壓，每天早上需要吃一顆血糖藥和血壓藥，請問你們的藥局有這些藥嗎？」
```

**重要：建議出國前，先把自己的病史和常用藥物名稱（中英文）寫在手機記事本裡，萬一需要就醫可以快速給醫生看。**

## 📝 回家練習

1. **必做**：下載並設定好 Google 翻譯 App，加入「繁體中文 ↔ 日語」和「繁體中文 ↔ 英語」的離線翻譯包。

2. **必做**：模擬一個情境：你在日本餐廳，服務生端來一碗湯，你不知道是什麼湯，請用拍照翻譯功能確認內容，並用日文問服務生「這個湯是什麼口味的？」

3. **選做**：把一篇你想去旅遊的國外網站文章，用 Google 翻譯或 ChatGPT 翻譯成中文，體驗閱讀外語資訊的輕鬆感。

## 🎯 本週重點

1. ✅ Google 翻譯支援拍照即時翻譯，是出國旅遊的最佳幫手
2. ✅ 出國前務必下載離線翻譯包，避免國外沒有網路
3. ✅ 機場、飯店、餐廳、問路每個情境都可以用 AI 翻譯輔助
4. ✅ 緊急就醫時可以透過翻譯清楚表達自己的症狀和病史
5. ✅ 把常用的翻譯語句預先存在手機裡，臨時需要時可以直接拿給外國人看

# 第7週：LINE 個人貼圖
日期：10/16（週五）｜授課時數：3小時

## 📖 課程摘要

### 為什麼要製作 LINE 個人貼圖？

相信大家使用 LINE 都會用到表情貼（貼圖），有時候覺得現成的貼圖不夠有趣、不夠貼近自己，或是想要有專屬於自己和家人的表情包。現在只要利用手機或平板，就可以輕鬆製作出屬於自己的 LINE 貼圖，而且上架到 LINE Creators Market 後，全世界的人都可以下載使用。

### 本週目標

本週我們要完成兩件事：
1. 學會用「LINE Creators」平台製作並上架個人貼圖
2. 了解 LINE 貼圖的規範，避免送審被退件

### 什麼是 LINE 個人貼圖？

LINE 個人貼圖是一種可以在 LINE 聊天時傳送的表情圖片。目前 LINE 官方開放讓一般人申請製作，審核通過後就能上架。有些人做自己孫子的表情包，有些人是寵物的萌照，也有人把自己畫成卡通角色，都是非常受歡迎的貼圖。

### LINE 貼圖的類型

LINE 貼圖主要有兩種形式：
1. **一般貼圖**（Static Sticker）：靜態圖片，最常見、最容易製作
2. **動態貼圖**（Animated Sticker）：會動的圖片，需要更多時間和技術

本週我們先學會製作一般靜態貼圖。

## 🛠️ 實作步驟

### 步驟1：準備貼圖素材

**LINE 貼圖的格式要求：**
- 數量：8張、16張、24張或32張（必須是8的倍數）
- 圖片尺寸：每張 370 × 320 像素（px）
- 檔案格式：PNG 或 JPG
- 檔案大小：每張不超過 1 MB

**老師建議：**
一開始不需要準備 16 張，可以先準備 8 張就好，這樣負擔比較輕。

**素材來源建議：**
以下任何一種都可以做成 LINE 貼圖：
- 自己的自拍照（使用美圖秀秀修飾過）
- 孫子或家人的有趣表情
- 寵物的照片
- 自己的手繪卡通（可以掃描或拍照）
- 風景或物品照片

**重要提醒：**
如果使用他人或名人照片做貼圖，可能會有肖像權問題。建議使用自己的照片或自己拍的東西，這樣最安全。

（請見課堂截圖：LINE 貼圖示意圖）

### 步驟2：用美圖秀秀裁切並修飾照片

LINE 貼圖需要特定的尺寸（370×320），所以需要先裁切好。

**操作步驟：**
1. 開啟美圖秀秀，選擇一張要製作成貼圖的照片
2. 點「剪裁」或「裁切」
3. 在比例選項中選擇「自訂」或「1:1」
4. 輸入尺寸：寬度 370，高度 320（或等比例放大到最大）
5. 移動照片位置，選擇臉部或主體放在畫面中央
6. 確認儲存

**一鍵去背（可選）：**
如果想要透明背景，可以：
1. 在美圖秀秀使用「AI 去背」功能
2. 去背後的背景會變成灰白格狀，代表透明
3. 以 PNG 格式儲存

### 步驟3：申請 LINE Creators 帳號

LINE 個人貼圖需要透過 LINE Creators 平台申請。申請流程如下：

**申請條件：**
- 必須有 LINE 帳號（大家應該都有）
- 年滿 18 歲（未滿 18 歲需要家長同意）

**申請步驟：**
1. 在手機的 LINE 主頁，點「設定」（齒輪圖示）
2. 往下滑，點「LINE Creators Market」
3. 點「開始申請」
4. 填寫以下資訊：
   - 暱稱（公開顯示的名字）
   - 電子郵件（用來收審核通知）
   - 設定銷售價格（台灣地區建議設 30-50 元，或選擇「免費」先試看看）
5. 閱讀並同意使用條款
6. 提交申請

**等待審核：**
申請送出後，LINE 需要幾個工作天來審核帳號。審核通過後就可以上架貼圖了。

### 步驟4：上傳並提交個人貼圖

帳號申請通過後，就可以開始上傳貼圖了：

**上傳步驟：**
1. 進入 LINE Creators Market 管理頁面
2. 點「新建作品」
3. 填寫貼圖資訊：
   - **名稱**：你的貼圖包叫什麼名字（例如「阿美的生活大小事」）
   - **描述**：簡單介紹這個貼圖包的內容（40字以內）
   - **標籤**：可以寫「生活」「家庭」「可愛」等關鍵字
4. 上傳所有準備好的圖片（8張、16張或其他數量）
   - 確認每張圖片都是 370×320 像素
5. 上傳一張「主圖」（顯示在 LINE 貼圖商店列表的封面）
6. 上傳一張「主頁橫幅」（寬度 1500 像素，高度 843 像素）
7. 點「預覽」確認所有內容正確
8. 點「提交」送出審核

### 步驟5：了解審核常見被退件的原因

LINE 對貼圖內容有嚴格規範，以下是最常見的退件原因，請特別注意：

1. **圖片尺寸不符**：嚴格要求 370×320 像素
2. **有太多留白**：圖片主體不能太小，要填滿大部分畫面
3. **版權問題**：使用明星、卡通角色、電影場景等有版權的圖片
4. **不雅內容**：涉及裸露、暴力、歧視等內容
5. **辨識度太低**：多張貼圖太相似，沒有明顯差異

**老師建議：**
- 先製作 8 張貼圖送出審核，熟悉流程後再慢慢增加到 16 張或更多
- 貼圖內容要有明顯差異，例如不同的表情或動作

### 步驟6：上架後如何宣傳

審核通過後（約 5-7 個工作天），你的貼圖就會出現在 LINE 貼圖小舖了！

**宣傳方式：**
1. 把連結分享到 LINE 群組和家人
2. 請親朋好友幫忙下載和評價
3. 可以在 Facebook 或其他社群分享

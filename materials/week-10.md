# 第10週：提示詞魔法
日期：11/6（週五）｜授課時數：3小時

## 📖 課程摘要

### 什麼是「提示詞」？

之前幾週，我們一直在跟 AI 說話和下指令，但有沒有發現：有時候 AI 回覆的內容正是我們想要的，但有時候 AI 給的答案卻讓人失望？這中間的差異，關鍵就在於「提示詞」（Prompt）。

提示詞就是你和 AI 對話時所輸入的文字。同一個問題，用不同的方式問，AI 的回答可能會有很大的不同。學會如何問對問題，是讓 AI 發揮最大效用的關鍵。

### 為什麼提示詞這麼重要？

想像你走進一家餐廳，如果你只跟服務生說「我想要一些吃的」，服務生可能會給你各種各樣的食物。但如果你說「我今天是素食者，請給我一份不需要肉、有蛋白質、適合晚餐的菜單推薦」，服務生給你的答案就會精準得多。AI 也是一樣——給的資訊越多、越清楚，AI 的回答就越符合需求。

## 🛠️ 實作步驟

### 步驟1：認識好提示詞的基本要素

好的提示詞通常包含以下四個要素：

**1. 角色（Role）**
告訴 AI 你希望它扮演什麼角色。
- 好的例子：「你是一位有10年經驗的營養師」
- 不好的例子：「我想要健康」（太模糊）

**2. 任務（Task）**
清楚地說明你要 AI 做什麼。
- 好的例子：「請為我設計一份為期一週的減肥食譜」
- 不好的例子：「減肥要吃什麼」（範圍太廣）

**3. 格式（Format）**
說明你希望 AI 如何呈現答案。
- 好的例子：「請用表格呈現，包含熱量、食材、準備時間」
- 不好的例子：「給我一些資訊」（不知道要什麼格式）

**4. 範例（Example）**
給 AI 一個具體的例子，幫助它理解你要的風格。
- 好的例子：「像這樣的格式：[示例內容]」
- 不好的例子：「按照一般的格式」（沒有具體範例）

### 步驟2：壞提示詞與好提示詞的對比演練

讓我們用實際例子比較不同提示詞的效果。

**情境：你想查詢如何種植番茄**

**❌ 壞提示詞：**
```
怎麼種番茄？
```

AI 可能只會回覆很基本、很短的資訊，缺少實用細節。

**✅ 好提示詞：**
```
你是一位有20年經驗的台灣有機農夫，擅長在家陽台種菜。
請告訴我如何在台北的公寓陽台，用盆栽種植小番茄。
請用以下格式回答：
1. 需要準備的工具和材料（附建議品牌）
2. 種植步驟（每個步驟詳細說明）
3. 多久澆水一次
4. 病蟲害防治方法
5. 預計多久可以收成
請用繁體中文回答，回答內容要詳細，適合完全沒有種菜經驗的初學者。
```

兩相比較，好提示詞給的答案會詳細得多！

### 步驟3：5種常見提示詞錯誤

**錯誤一：問題太模糊**
- ❌：「天氣怎麼樣？」
- ✅：「今天台北（11月6日）的天氣如何？最高溫度和最低溫度是多少？適合出門嗎？」

**錯誤二：一次問太多問題**
- ❌：「告訴我關於糖尿病的一切，包括飲食禁忌、運動建議、用藥注意事項」
- ✅：每次只問一個主題，問完再問下一個

**錯誤三：忘記說對象是誰**
- ❌：「這個東西怎麼吃？」
- ✅：「我80歲的母親牙齒不好，消化功能也比較弱，請告訴我她適合吃什麼食物和怎麼吃」

**錯誤四：沒有指定答案長度**
- ❌：「介紹一下日本」
- ✅：「請用200個字以內，介紹日本東京適合60歲以上長輩的旅遊景點」

**錯誤五：沒有說明用途**
- ❌：「寫一首生日快樂歌」
- ✅：「寫一首給80歲長輩的生日快樂歌，風格溫馨感人，要提到他一生對家庭的付出，適合在壽宴上由孫子合唱」

### 步驟4：10個好用提示詞範例

以下是老師精選的 10 個日常實用提示詞，請學員複製貼上到 ChatGPT 試試看：

**1. 食譜查詢：**
```
我是一個完全不會做菜的人，請給我一道「紅燒魚」的詳細食譜，包括：所需食材（含分量）、所有步驟（每步附圖片說明位置）、烹飪時間、小撇步。台灣家常口味。
```

**2. 健康諮詢：**
```
我今年65歲，最近膝蓋爬樓梯時會酸痛，請告訴我可能的原因，以及日常生活中如何保養和舒緩。不需要就醫的話，給我飲食和運動建議。
```

**3. 旅遊規劃：**
```
我要帶80歲的父親去花蓮三天兩夜，他走路不方便（需要拐杖），不喜歡人多的地方。請幫我規劃行程，每個景點說明需要走多少路、有無廁所、能否讓輪椅通過。
```

**4. 節慶祝福：**
```
我要寫中秋節祝福語給我的岳父岳母，內容要提到「月亮圓、家庭也圓滿」的意象，語氣真誠感人，120個字以內，不要太商業化。
```

**5. 法律問題：**
```
我家的土地被邻居占用了一部分（約2坪），對方說那是他們的地。請用白話的方式，告訴我接下來可以怎麼處理，以及需要準備什麼文件。
```

**6. 家電使用：**
```
我的新洗衣機是國際牌的滾筒洗衣機，請用最簡單的方式，告訴我如何洗羊毛衣物，要調什麼模式、水溫多少、要用什麼洗劑。
```

**7. 投資理財：**
```
我65歲，存款大約200萬，想要保守投資，請告訴我適合的投資方式有哪些，風險最低的是什麼，不要推薦股票或複雜的金融商品。
```

**8. 書信寫作：**
```
我要寫一封正式的書信給區公所，申請住宅補貼，請幫我寫一篇350字以內的內容，口氣要恭敬但不失尊嚴，內容要包含：申請人身分說明、申請原因、具體訴求。
```

**9. 旅遊英文：**
```
我要去美國自助旅遊，英文不太流利，請幫我列出30句最實用的機場英文句子，包含：機場報到、行李遺失、通關、登機門等情境，每句附上中文和羅馬拼音。
```

**10. 興趣學習：**
```
我是一位60歲的初學者，想要學習書法（毛筆字），請幫我列一張需要購買的工具清單（附建議價格區間），以及從哪個字開始練習最好，並說明原因。
```

### 步驟5：迭代優化技巧

如果 AI 的回答不夠好，可以這樣問：

**追問改進：**
「這個答案不夠詳細，請再補充更多具體例子」

**要求更簡單：**
「請用更白話的方式說明，不要用太多術語」

**要求不同角度：**
「請從另一個角度回答這個問題」

1. ✅ 好的提示詞要具備四要素：角色+任務+格式+範例

3. ✅ 指定答案長度和格式，讓 AI 的回覆更有用

# 第4週：AI 社交應用：祝福語、防詐
日期：9/25（週五）｜授課時數：3小時

## 📖 課程摘要

### AI 如何成為你的社交小幫手？

現代人生活中有許多需要「表達心意」的場合：春節、中秋、端午要發祝福簡訊；生日、母親節、父親節要寫卡片；孫子出生、朋友結婚要送祝賀詞。很多人常常絞盡腦汁想不出適合的話，或者每次都只會寫「佳節愉快」四個字，顯得沒有誠意。本週我們要學習如何用 AI 幫你寫出真誠動人的祝福語。

### AI 祝福語的三大優勢

第一，**量身訂做**：只要告訴 AI 你要祝福的對象、場合、你們的關係，AI 就能寫出非常個人化的內容，而不是制式的罐頭訊息。

第二，**多樣風格**：你可以要求 AI 寫文言文、白話文、押韻詩句、順口溜，甚至模仿某個名人的風格。

第三，**省時省力**：30秒就能得到一份完整的祝福語，不需要花半小時想破頭。

### 常見需要祝福語的場合

以下是我們本週會練習的各種場合：
- 春節（農曆新年）
- 中秋節
- 端午節
- 生日（自己、配偶、子女、朋友）
- 母親節 / 父親節
- 孫子出生
- 朋友結婚或搬新家
- 升遷、退休祝賀

### LINE 防詐知識

本週也會教大家如何辨識手機上常見的 LINE 詐騙訊息，以及如何設定 LINE 的安全功能來保護自己。根據內政部警政署的統計，LINE 是目前最常被用作詐騙工具的通訊軟體。常見的 LINE 詐騙手法包括：假冒親友借錢、假冒銀行或政府機關、投資獲利訊息、假購物網站連結、假的投票或優惠活動。

## 🛠️ 實作步驟

### 步驟1：用 AI 撰寫春節祝福語

打開 ChatGPT，輸入以下句子：

```
我要寫 LINE 春節祝福給我的子女和孫子，風格要溫馨感人，內容要提到珍惜團聚的時光，總共150個字以內，請給我三個不同版本。
```

AI 會給你三個版本的祝福語。你可以選一個最喜歡的，或者把幾個版本合併成自己的版本。

**練習其他場合：**
```
我要寫母親節祝福給我的太太，她是我孩子的媽媽，風格要感性，提到她對家庭的付出，100個字以內，請給我兩個版本。
```

```
我要寫生日祝福給我的好兄弟，他今年60歲了，我們認識30年，風格要輕鬆幽默但不失真誠，80個字以內。
```

### 步驟2：用 AI 撰寫各種節慶祝福語

以下是各節慶的練習句，請分別輸入 AI，看看回覆的品質：

**中秋節：**
```
請幫我寫中秋節祝福語，給我的鄰居和社區朋友，語氣要親切溫暖，提到賞月和吃月餅，60個字以內。
```

**端午節：**
```
請幫我寫端午節祝福語，給我的家族群組，提到健康平安和吃粽子，語氣要溫馨，50個字以內。
```

**孫子出生祝賀：**
```
請幫我寫祝賀詞，給剛剛出生的孫子（男嬰），內容要提到祝福健康成長和未來光明，100個字以內，可以用文言文或詩句形式。
```

### 步驟3：LINE 訊息防詐演練

**練習辨識詐騙訊息：**
老師會在課堂上分享幾個真實的 LINE 詐騙訊息範例，請跟著演練如何辨識：

1. **假親友借錢**：訊息說「爸，我在外面手機掉了，這是我的新帳號，請匯錢過來」→ 破解方法：打電話給子女確認，不要直接匯款

2. **投資獲利**：訊息說「加入 LINE 群組，每天都有老師帶你操作，保証獲利20%」→ 破解方法：這是典型的投資詐騙，果斷封鎖並檢舉

3. **假銀行通知**：訊息說「您的帳戶有異常，請點連結重新驗證」→ 破解方法：銀行不會用 LINE 要求你點連結，應直接致電銀行客服

**設定 LINE 安全功能：**

請跟著老師一起設定以下功能：
1. 開啟「LINE Labs」的新功能測試
2. 設定「阻擋陌生人的訊息」
3. 關閉「允許被加入群組」的功能
4. 設定「登入許可」保護

（請見課堂截圖：LINE 安全設定的每一步操作說明）

### 步驟4：學會檢舉和封鎖

如果你收到可疑訊息：
1. 長按該訊息
2. 選「檢舉」
3. 選擇檢舉原因（廣告、詐騙、騷擾等）
4. 選擇是否封鎖該帳號

### 步驟5：建立你的祝福語資料庫

請 AI 幫你寫十種不同場合的祝福語（每種 1-2 句），把這些祝福語複製到 LINE Keep 筆記或手機記事本，以後要用的時候，直接打開複製貼上即可。

## 📝 回家練習

1. **必做**：請用 AI 幫你寫一個「朋友搬新家」的祝賀語，然後把這段話 LINE 給你身邊有搬家經驗的朋友。

2. **必做**：檢查自己手機上的 LINE 設定，確認已經開啟「阻擋陌生訊息」功能。如果不會設定，請下週上課時舉手詢問老師。

3. **選做**：把你最喜歡的三段祝福語收藏到 LINE Keep，並練習在需要的時候快速找到並傳送。

## 🎯 本週重點

1. ✅ AI 可以幫你寫各種場合的個人化祝福語，不需要再想破頭
2. ✅ LINE 詐騙常見手法包括假親友借錢、投資獲利誘惑、假銀行連結
3. ✅ 遇到可疑訊息，牢記「不點連結、不匯款、先求證」的原則
4. ✅ 善用 165 防詐騙專線，有疑慮就打電話詢問
5. ✅ 把常用祝福語整理成資料庫，需要時快速取出使用

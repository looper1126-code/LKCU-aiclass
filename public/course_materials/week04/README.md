## 課堂教學步驟

### 第一節：AI 幫寫祝福語（19:00 - 19:50）

#### 👨🏫 教學目標
學會請 AI 幫你寫出有誠意的祝福，不再只會說「生日快樂」！

#### 💬 祝福語問法教學

**場合不同，問法不同：**

| 場合 | 問法 |
|:---|:---|
| 生日 | 「幫我寫一段生日祝福，要感人的」 |
| 母親節 | 「母親節祝福媽媽，四句話以內」 |
| 過年 | 「新年祝福長輩，四字成語」 |
| 父親節 | 「父親節卡片文案，簡短有力」 |
| 結婚 | 「結婚祝賀語，要莊重」 |
| 畢業 | 「畢業祝福同學，活潑一點」 |

**風格選擇：**
- 正式 ✅：「幫我寫一段正式的祝福”
- 俏皮 😄：「幫我寫俏皮的祝福」
- 感人 ❤️：「幫我寫感人的祝福」
- 搞笑 🎉：「幫我寫逗趣的祝福」

#### 📱 實作範例

**案例 1：媽媽生日**
> Input: 「幫我寫給媽媽的生日祝福，要感人」
> Output: 
> ```
> 親愛的媽媽，
> 感謝您一路的養育恩情，
> 願您天天開心，
> 健康長壽。
> 永遠愛您 的孩子敬上
> ```

**案例 2：朋友結婚**
> Input: 「朋友結婚，祝賀語」
> Output:
> ```
> 百年好合，永結同心
> 早生貴子，白頭偕老
> 珠聯璧合，，美滿良緣
> ```

**案例 3：過年紅包訊息**
> Input: 「過年給長辈的祝賀詞」
> Output:
> ```
> 恭喜發財、大吉大利
> 身體健康、萬事如意
> 龍馬精神、富貴滿堂
> ```

### 第三節：防詐騙識別（21:00 - 21:50）

#### 👨🏫 教學目標
學會辨識可疑訊息與詐騙手法，保住辛苦錢！

#### 🚨 常見詐騙手法

| 手法 | 特徵 | 怎麼做 |
|:---|:---|:---|
| 假冒親友借錢 | 聲稱換電話 | 電話確認！ |
| 假投資群組 | 穩賺不賠 | 立馬退群！ |
| 釣魚連結 | 要求填資料 | 不點擊！ |
| 假冒官署 | 說你欠稅 | 165 反詐！ |
| 猜拳詐騙 | 剪刀石頭布 | 不玩！ |

#### 📱 AI 教你防詐

> 「這個訊息是詐騙嗎？幫我分析」

> 「這則 LINE 訊息可以相信嗎？」

> 「[附上截圖] 這是不是詐騙？」

#### 💡 防詐小技巧

1. **不貪**：穩賺不賠都是假的
2. **不急**：逼你趕快匯款是騙局
3. **不點**：不明連結不要點
4. **不回**：陌生人訊息別亂回
5. **不通**：借錢一定要打電話確認

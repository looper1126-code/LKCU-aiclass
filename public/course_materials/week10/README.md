## 課堂教學步驟

### 第一節：什麼是最好的 Prompt（19:00 - 19:45）

#### 👨‍🏫 教學目標
了解為什麼同一個問題，問法不同，結果差很多。

#### 📖 教學重點

**Prompt 就是「對 AI 說的話」**

#### ❌ 錯誤示範
```
「推薦好吃的」
```
→ AI 不知道你要推薦什麼

#### ✅ 正確示範
```
「台北市大安區，
捷運站附近，
平價美味的火鍋店推薦，
適合2-3人用餐」
```
→ AI 很清楚你要什麼

#### 💡 黄金公式

**好的 Prompt = 角色 + 任務 + 細節**

```
【我是誰】要請你幫我
【要做什麼】具體說清楚
【有什麼限制】越多越好
【Output格式】要表格?列表?文章?
```

### 第二節：五大 Prompt 技巧（19:45 - 20:30）

#### 👨‍🏫 教學目標
學會五種實用的提問技巧。

#### 📝 五大技巧

**1️⃣ 明確具體**
| 問錯 | 問對 |
|------|------|
| 天氣怎麼樣 | 今天台北的天氣如何？溫度？ |
| 推薦餐廳 | 大安區高CP值餐廳 |

**2️⃣ 指定格式**
- 「用表格呈現」
- 「列點說明」
- 「給我500字以內」

**3️⃣ 提供情境**
- 「我是第一次出國」
- 「家裡有小朋友」
- 「預算有限」

**4️⃣ 分步驟問**
- 先問大的
- 再問細的
- 逐步缩小范围

**5️⃣ 讓 AI 扮演角色**
```
「請你扮演专业的营养师，
帮我规划一周的健康菜单」
```

#### 💬 課堂練習
每个人把自己之前的失敗經驗拿出來改寫：
- 加入角色
- 加入細節
- 指定格式

### 第三節：Template 提示詞庫（20:30 - 21:15）

#### 👨‍🏫 教學目標
學會使用模板，快速上手。

#### 📋 常用模板

**📝 食譜模板**
```
我想做【菜名】，
材料有【列出材料】，
請給我【詳細/簡易】的做法，
要在【多少時間】內完成。
```

**✈️ 旅遊模板**
```
我要去【地點】，
【天數】天【夜】，
一個人預算【多少元】，
同行者有【誰】，
請帮我安排【每天的】行程。
```

**💬 祝福模板**
```
【節日】要到了，
我要送給【誰】，
【風格】的祝福語，
【字數】字以內。
```

**📊 学习模板**
```
我不懂【主題】，
請用【國中生/小朋友】听得懂的方式解释，
举三个例子。
```

## 常見問題 FAQ

### Q1：Prompt 越長越好嗎？
**A**：適度就好，重點是有用資訊，不是廢話。

### Q2：要一次問還是分開問？
**A**：複雜問題分開問，簡單問題一次問。

### Q3：AI 忘記之前說的？
**A**：如果對話太長，可能會遺忘。可以說「回顧一下之前說的...」

### Q4：要一直被拒絕怎麼辦？
**A**：換個問法，或分几步问。

### Q5：Prompt 的極限？
**A**：沒有標準答案，多練習就會找到感覺。

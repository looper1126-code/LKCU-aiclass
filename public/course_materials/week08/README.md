## 課堂教學步驟

### 第一節：初識 Suno（19:00 - 19:45）

#### 👨‍🏫 教學目標
認識 AI 音樂生成工具 Suno。

#### 📱 什麼是 Suno？

**Suno**：AI 音樂創作平台
- 只要打字描述，AI 就會作曲
- 支援中文歌詞
- 免費版本可用

**特色**：
- 🎵 任何音樂類型
- 🎤 有人聲演唱
- 🎸 完整編曲

**類似服務**：
- Udio
- MusicGen

#### 🎧 範例聆聽
老師播放幾個範例：
- 抒情歌｜饒舌｜電子

---

### 第二節：帳號與基本操作（19:45 - 20:30）

#### 👨‍🏫 教學目標
學會注册 Suno 並生成第一首歌。

#### 📱 註冊步驟

**電腦版：**
1. 打開 suno.ai
2. 點「Sign Up」
3. 用 Google 帳號或 Email 註冊
4. 驗證 Email
5. 開始創作

**手機版：**
1. 到 suno.ai（瀏覽器）
2. 行動版網頁

#### 💡 生成音樂公式

```
Style/Genre：【曲風，如抒情流行、R&B、搖滾】
Mood：【心情，如溫暖憂傷、快樂積極】
Subject：【主題，如懷念故鄉、追求夢想】
```

**範例 prompt：**
```
Style: 抒情流行抒情歌
Mood: 温暖感人
Subject: 感謝身邊的人
Language: 中文
```

### 第三節：歌詞創作（20:30 - 21:15）

#### 👨‍🏫 教學目標
學會寫讓 AI 唱的中文歌詞。

#### 📝 歌詞技巧

**選填模式：**
1. 點「Custom Mode」
2. 輸入你的歌詞
3. 選曲風
4. 點 Generate

**簡單歌詞公式：**
```
[Verse 1]（主歌1）
第一段verse寫甚麼...
[Verse 2]（主歌2）
第二段verse來了...
[Chorus】
副歌高潮歌詞！
[Outro】
結尾部份～
```

#### ✍️ 簡單歌詞範例

```
[Verse]
今天的陽光很温暖
想起小時候的玩伴
我們一起長大
[Phook]
朋友啊謝謝你
有你有我有希望
我們永永遠遠
```

#### ⚠️ 注意
- 避免太長
- 避免有版权的词（比如其他歌的歌词）

## 常見問題 FAQ

### Q1：音樂版權是誰的？
**A**：Suno 生成的音乐可用于个人用途，商业使用有限制。

### Q2：為什麼生成不出來？
**A】：檢查网络連線，或者换个描述试试。

### Q3：聲音不��人聲？
**A】：AI 學習的結果就是這樣，有時候會有點生硬。

### Q4：可以指定歌手嗎？
**A】：無法指定，但可以說風格像某類歌手。

### Q5：為什麼要付費？
**A】：付費版本可以生成更多首，且無廣告。

## 課堂教學步驟

### 第一節：ChatGPT Image 基礎（19:00 - 19:45）

#### 👨‍🏫 教學目標
了解 ChatGPT Image 功能。

#### 📱 如何使用

**功能位置**：
- ChatGPT Plus 中的 DALL·E
- 或使用 GPT-4o

**使用方法**：
1. 打開 ChatGPT
2. 切換到 GPT-4o (with Canvas)
3. 直接描述想要的圖片

**費用**：
- Plus 用戶：每週 15 個配額
- Free 用戶：有限次數

---

### 第二節：生活實用類（19:45 - 20:30）

#### 👨‍🏫 教學目標
學會產生生活應用的圖片。

#### 🍽️ 範例 1：菜單設計

**Prompt**：
```
請幫我製作一張文青風咖啡館菜單，版面包含飲品、甜點、價格與簡單插畫。使用奶油色背景、手寫字體、咖啡杯與植物元素
```

**應用場景**：
- 家族聚會邀請卡
- 旅遊回憶冊
- 食譜卡

---

#### 🗓️ 範例 2：季節月曆

**Prompt**：
```
請幫我生成一張秋季主題月曆設計，畫面包含月份日期排版、楓葉、柔和天空與可愛小插圖。整體使用橘色、米白與淺棕色
```

**應用場景**：
- 手機桌布
- 桌曆製作
- 節日卡片

---

#### ✈️ 範例 3：旅遊海報

**Prompt**：
```
幫我生成一張巴黎旅遊雜誌拼貼風海報，風格：法式插畫+高級旅遊雜誌感，水彩手繪風格
```

**應用場景**：
- 家族旅遊紀念冊
- 旅遊分享圖文

---

### 第三節：靈感創意類（20:30 - 21:15）

#### 🎯 範例 4：LINE 貼圖

**Prompt**：
```
請幫我設計一組 Q 版角色貼圖，角色表情誇張可愛，包含開心、無奈、生氣與驚訝四種情緒，風格簡單乾淨、白底
```

**應用**：這個跟第7週的 LINE 貼圖課程呼應

---

#### 🖼️ 範例 5：Y2K 風格照片轉換

**Prompt**：
```
將這張照片轉換成 Y2K 風格：螢光色系、幾何愛心、復古膠片質感、暖色調顆粒感
```

**應用場景**：
- 社群發文
- 照片回憶冊

---

#### 📊 範例 6：簡報封面

**Prompt**：
```
設計一張 AI 趨勢論壇簡報封面，主題：未來辦公室。風格：科技感、幾何線條、深色背景。包含標題與日期
```

**應用**：
- 商務簡報
- 產品發表會

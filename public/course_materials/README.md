# AI 智慧生活應用｜13週完整教材

> 這是 Jerry Kao 老師在林口社區大學開立的「AI智慧生活應用」課程教材，專為樂齡族設計，無需科技背景，只要會用手機就能學會讓 AI 成為生活好幫手。

## 🎯 課程目標

本課程結束後，學員將能夠：

1. ✅ 熟練使用 ChatGPT 和 Gemini 與 AI 溝通
2. ✅ 用 AI 查詢食譜、規劃旅遊行程
3. ✅ 使用 AI 修圖工具編輯照片
4. ✅ 製作個人 LINE 貼圖
5. ✅ 創作 AI 音樂
6. ✅ 生成 AI 圖片
7. ✅ 使用 AI 翻譯功能出國旅遊
8. ✅ 建立安全的上網習慣
